# IWC SC70: Cetacean culture
Used to search reports from the IWC SC and IWC CC for key terms relevant to cetacean culture

Two scripts: one for files with one-column format (2019-2024) the other for two-column format (2015-2018)

All files downloaded from IWC archives: https://archive.iwc.int/pages/collections_featured.php?

NB: some reports need their title and blank pages removed in order to work
