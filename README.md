# IWC SC70: Cetacean culture
We searched IWC Scientific Committe reports and IWC Conservation Committee reports for key terms relevant to cetacean culture

## About the scripts
Find Keywords (1 col): for files with one-column format (2019-2024) 

Find Keywords (2 col): for files with two-column format (2015-2018)

## About the data
All files downloaded from IWC archives: https://archive.iwc.int/pages/collections_featured.php?

NB: some PDFs have formatting issues that cause the scripts to fail or return errors
